# Quantile_Regression_CMI_WS
Notebooks, slides, code and resources for a hands-on workshop on quantile regression in the social sciences.

# Beyond the Mean: Quantile Regression for Distributional Analysis in Social Science Research

This repository contains the teaching materials, notebooks, slides, datasets, and supporting resources developed for [the Cathie Marsh Institute for Social Research (CMI) 2026 Short Course](https://www.cmi.manchester.ac.uk/connect/training/):

**Beyond the Mean: Quantile Regression for Distributional Analysis in Social Science Research**

The course introduces quantile regression as a method for analysing heterogeneous and distributional effects in social science research. Participants learn the conceptual foundations of quantile regression, estimation and inference, practical implementation in R, and interpretation of results using real-world datasets.

The repository includes:

- Quarto notebooks
- Slide decks
- R code and exercises
- Supporting readings and references

The materials were developed for the CMI Training Programme at The University of Manchester and are intended to support teaching, learning, and reproducible quantitative research.
